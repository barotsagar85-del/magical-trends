# Magical Trends — Natural Diamond Pricing Portal

This repository contains the interactive digital pricing portal for Magical Trends.

## Overview
The portal is designed for professional diamond supply chain presentations. It features:
- Interactive pricing grids for Round and Fancy shape diamonds.
- Company profile and compliance documentation.
- Live stock demonstration with certificate and video integration.
- Ethical sourcing transparency (Kimberley Process, AML/FATF compliant).

## Viewing the Portal
The portal is a self-contained HTML application. To view it, simply open `index.html` in any modern web browser.
